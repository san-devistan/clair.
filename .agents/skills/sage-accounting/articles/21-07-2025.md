## 21st July 2025
To ensure continued functionality and access to the latest features and improvements, we strongly encourage you to begin migrating your applications to API V3.1.
#### Why this matters:
API V3.1 offers enhanced performance, improved consistency, and better support for future development. After the deprecation date, API V3 will no longer be supported or accessible. Migration Support: To assist with your transition, we’ve prepared a comprehensive migration guide: Migrate from V3 to V3.1
If you have any questions or need help with the migration process, our developer support team is here to help. Please contact us at sagesoft.developers@sage.com for individual assistance.
